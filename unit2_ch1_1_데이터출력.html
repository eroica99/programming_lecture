1. 자료형
교과서 pp.43–46
자료형의 개념
p.43
자료형 (data type)
데이터의 종류와 특성에 따라 구분하는 이름이다. 프로그래밍을 통하여 입출력되는 다양한 데이터는 종류와 특성에 따라 다른 형태로 컴퓨터 내부에 저장·처리되므로, 정확하게 표현해야 한다.
컴퓨터 내부에서는 모든 데이터를 2진수(0 또는 1)로 저장·처리하며, 자료형에 따라 저장 방식이 달라진다.
자료형의 종류
pp.43–44 [표Ⅱ-1]
정수 (int)

소수점 아래에 작은 수가 없는 딱 떨어지는 값으로, 음의 정수, 0, 양의 정수를 의미한다.

사칙 연산이 서로 가능한 2진수 형태로 바뀌어 컴퓨터 내부에 저장된다.

파이썬 표현 예시: -1, 0, 1, 2, 100

문자열 (str)

문자를 1개 이상 순서대로 나열한 것이다. 문자 한 개는 아스키(ASCII) 또는 유니코드(unicode) 체계에 따라 특별한 코드로 변환되어 저장된다.

파이썬에서는 문자와 문자열을 따로 구분하지 않고, 문자도 문자열로 저장하여 처리한다.

파이썬 표현: 큰따옴표(") 또는 작은따옴표(')로 묶어 표현한다.

파이썬 표현 예시: "hello", "1", "123", 'a', 'A'

실수 (float)

소수점을 사용해 표현한 수로, 소수점 아래에 있는 작은 수까지 표현한다.

2진수 형태로 변환되어 저장되지만, 정수와는 다른 방식(IEEE-754 표준)으로 저장된다.

파이썬 표현: 소수점과 함께 부호와 숫자를 붙여 쓴다.

파이썬 표현 예시: 1.0, 1.414, -2.718, 3.14

논릿값 (bool)

수학자 조지 불(George Boole)이 만든 개념으로, 참(True) 또는 거짓(False) 두 가지 값만 가진다.

and, or, not과 같은 논리 연산으로 계산되며, 논리 연산의 결과도 논릿값이다.

일반적으로 참은 정숫값 1, 거짓은 정숫값 0으로 저장된다. 0이 아닌 모든 값은 참으로 계산한다.

파이썬 표현: True, False만 사용한다. (첫 글자 반드시 대문자)

선택 실행(if)과 반복 실행(while)과 같은 프로그램 실행 제어에 사용된다.

자료형설명파이썬 표현 예시
정수음의 정수, 0, 양의 정수-1, 0, 1, 100
문자열문자를 1개 이상 나열한 것"hello", 'A', "123"
실수소수점을 사용해 표현한 수1.0, -2.718, 3.14
논릿값참, 거짓True, False

※ 1(정수), "1"(문자열), 1.0(실수)은 각각 서로 다른 자료형이다. (p.45)

자료형 혼용 시 주의 사항
p.45
자료형이 다른 데이터를 서로 계산하거나 잘못 처리하면 프로그램 실행 중 오류가 발생하거나 예상과 전혀 다른 결과가 나타날 수 있다.
코드 예시결과이유
print(1 + "1") 오류 발생 정수와 문자열은 +로 계산할 수 없다.
print(3 * "hello") hellohellohello 문자열을 정숫값만큼 반복한 문자열이 만들어진다.
자료형 변환 함수
p.46 [표Ⅱ-3]

필요에 따라 데이터를 원하는 자료형으로 변환해야 할 때 사용하는 함수이다.

변환 함수변환 내용사용 예
int() 수 또는 문자열을 정숫값으로 변환한다. int("123") → 123
float() 수 또는 문자열을 실수로 변환한다. float("3.14") → 3.14
str() 어떤 값을 문자열로 변환한다. str(123) → "123"
bool() 수 또는 문자열을 논릿값으로 변환한다. bool(0) → False
chr() 정숫값을 유니코드 문자로 변환한다. chr(65) → 'A'
ord() 유니코드 문자를 정숫값으로 변환한다. ord('A') → 65

※ input()으로 입력받은 값은 항상 문자열이므로, 정수로 사용하려면 int(input())으로 변환해야 한다.

2. print() 함수
교과서 pp.47–51
기본 형식
p.47
print() 함수
주어진 값을 문자열로 화면에 출력하는 함수이다. 출력 후 기본적으로 새 줄(\n)로 이동한다.
형식설명
print(값)값을 출력한다.
print(값1, 값2, ...)여러 값을 스페이스를 사이에 넣어 순서대로 출력한다.
print(값1, 값2, sep="구분자")값 사이에 지정한 구분자를 넣어 출력한다.
print(값, end="끝문자")출력 후 새 줄 대신 지정한 끝 문자를 출력한다.
코드
print(123) print("Hello")
실행 결과
123 Hello
sep — 구분자 지정
pp.48–49

출력할 값을 쉼표(,)로 나열하면 기본적으로 스페이스를 사이에 넣어 출력한다. sep 매개변수로 구분자를 원하는 문자로 바꿀 수 있다.

매개변수기본값설명
sep " " (스페이스) 값과 값 사이에 넣을 구분 문자. ""로 설정하면 구분 없이 붙여 출력한다.
코드
print("Hello", "World") print(13, 45, 29, sep=":") print(13, 45, 29, sep="") print("A", "B", "C", sep=",")
실행 결과
Hello World 13:45:29 134529 A,B,C

※ ""(빈 문자열)은 아무 문자도 없다는 의미이다.

end — 끝 문자 지정
pp.50–51

print()는 출력 후 기본적으로 새 줄(\n)로 이동한다. end 매개변수로 출력 후 끝 문자를 원하는 문자로 바꿀 수 있다.

매개변수기본값설명
end "\n" (새 줄) 값을 모두 출력한 후 마지막에 출력할 문자. ""로 설정하면 새 줄 없이 이어서 출력한다.
코드
print("A", end=",") print("B", end=",") print("C")
실행 결과
A,B,C

※ print("A", "B", "C", sep=",")와 동일한 결과이다.

이스케이프 시퀀스 (escape sequence)
p.50

백슬래시(\)와 다른 문자를 함께 사용하여 특별한 처리나 뜻을 표현하는 표기법이다.

표기의미사용 예
\n 새 줄로 바꿈 (new line) print("Hello\nWorld") → Hello와 World가 줄 바꿈되어 출력
\' 작은따옴표 문자 print('it\'s') → it's
\" 큰따옴표 문자 print("\"Hello\"") → "Hello"
\\ 백슬래시 문자 print("\\") → \

※ 백슬래시(\)와 슬래시(/)는 서로 다른 문자이다. 키보드에 따라 ₩로 표시되기도 한다.

sep="\n" 활용 예
print("Hello", "World", sep="\n")
실행 결과
Hello World

매개변수적용 위치기본값변경 예시
sep 값과 값 사이 " " (스페이스) sep=":", sep=",", sep=""
end 출력 마지막 "\n" (새 줄) end=",", end=" ", end=""

문제 4
다음 중 자료형과 print() 함수에 대한 설명으로 옳지 않은 것은?
① 파이썬에서 논릿값은 True와 False만 사용하며 첫 글자는 대문자이다.
② 파이썬에서 문자와 문자열은 따로 구분하지 않고 모두 문자열로 처리한다.
③ print() 함수의 sep 기본값은 스페이스(" ")이고, end 기본값은 새 줄("\n")이다.
④ 정수 1과 문자열 "1"과 실수 1.0은 모두 같은 자료형이다.
⑤ \n은 출력할 위치를 새 줄로 바꾸는 이스케이프 시퀀스이다.
정답: ④
교과서 p.45 참고
1(정수), "1"(문자열), 1.0(실수)은 각각 서로 다른 자료형이다. 컴퓨터 내부에서 각각 다른 형태로 저장·처리된다. 자료형이 다른 데이터를 혼용하면 오류가 발생하거나 예상치 못한 결과가 나타날 수 있다.
