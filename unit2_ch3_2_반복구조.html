1. while ~ 반복 구조
교과서 pp.109–111
형식과 동작 방식
p.109
반복 구조
특별한 조건이나 상황에 따라 특별한 작업이나 처리를 반복해서 실행하는 구조이다.
형식
while 조건식: 실행 1 실행 2
if 와의 차이
if: True이면 한 번만 실행 while: True인 동안 계속 반복 조건이 False가 되면 반복 종료
무한 루프 주의: 반복 조건이 절대 False가 되지 않으면 프로그램이 영원히 실행된다. 반복 블록 안에서 반드시 조건을 변화시키는 코드(예: i = i + 1)가 있어야 한다.
예시 — 1부터 n까지 출력
p.110
코드
n = int(input()) i = 1 while i <= n: print(i) i = i + 1
입력
3
결과
1 2 3
반복 횟수i 값 (진입 시)조건 i<=3출력i 값 (실행 후)
1회1True12
2회2True23
3회3True34
종료4False——
예시 — 별 n개 한 줄 출력
p.111
코드
n = int(input()) i = 1 while i <= n: print("*", end="") i = i + 1
입력
5
결과
*****

※ end=""로 줄 바꿈 없이 이어서 출력

2. for ~ 반복 구조와 range()
교과서 pp.112–116
형식과 range() 함수
p.112
형식
for 변수 in 수열/리스트: 실행 1 실행 2
while과의 차이
while: 조건이 True인 동안 반복 → 횟수 제한 없음 for: 수열/리스트의 개수만큼만 반복 → 횟수가 정해져 있음

range() 함수는 일정한 차이값을 가진 정수 수열을 만든다.

형식의미생성 수열 예시
range(n) 0부터 n-1까지 1씩 증가 range(3) → 0, 1, 2
range(a, b) a부터 b-1까지 1씩 증가 range(1, 4) → 1, 2, 3
range(a, b, k) a부터 b 미만, k씩 증가(또는 감소) range(1, 10, 2) → 1, 3, 5, 7, 9
range(5, -1, -1) → 5, 4, 3, 2, 1, 0
range(a, b)에서 b는 포함되지 않는다. 1부터 n까지 반복하려면 range(1, n+1)을 사용해야 한다.
예시 — 1부터 n까지 출력 (while과 비교)
p.113
while 방식
n = int(input()) i = 1 while i <= n: print(i) i = i + 1
for 방식
n = int(input()) for i in range(1, n+1): print(i)

※ 두 코드 모두 같은 결과. for 방식이 더 간결하다.

예시 — 리스트 순회
p.114
코드
t = input().split() for s in t: print(s)
입력
hello python world
결과
hello python world
예시 — 리스트 역순 출력 + append()
pp.115–116
코드 (역순)
d = list(map(int, input().split())) n = len(d) for i in range(n-1, -1, -1): print(d[i])
입력
1 2 3 4 5
결과
5 4 3 2 1

※ range(n-1, -1, -1) → n-1, n-2, ..., 1, 0 순서 (감소)

코드 (append로 입력)
n = int(input()) d = [] for i in range(n): d.append(int(input())) for i in range(n): print(d[i], end=" ")
입력
3 10 20 30
결과
10 20 30

※ d = [] — 빈 리스트 생성  ·  d.append(값) — 리스트 마지막에 값 추가

3. continue와 break
교과서 pp.117–118
continue — 다음 반복으로 건너뜀

반복 실행 도중 continue가 실행되면 반복 블록의 남은 부분을 실행하지 않고 다음 반복으로 건너뛴다. 반복 자체는 계속된다. (p.117)

코드 (3의 배수 제외)
n = int(input()) for i in range(1, n+1): if (i % 3) == 0: continue print(i, end=" ")
입력
10
결과
1 2 4 5 7 8 10

※ i=3, 6, 9일 때 continue → print 건너뜀. 나머지는 정상 출력

break — 반복 전체 중단

반복 실행 도중 break가 실행되면 반복 블록의 남은 부분과 남은 모든 반복이 중단된다. 반복문을 완전히 빠져나온다. (p.118)

코드 (음수 입력 시 중단)
while True: n = int(input()) if n < 0: break print(n)
입력
3 0 -2
결과
3 0

※ while True: — 조건식이 항상 True이므로 break 없이는 무한 반복

continue vs break 비교
pp.117–118
명령동작반복 계속 여부사용 상황
continue 현재 반복의 나머지 코드 건너뜀, 다음 반복으로 이동 계속됨 특정 경우만 건너뛰고 싶을 때
break 반복 블록 남은 코드 + 남은 모든 반복 중단 중단됨 특정 조건에서 반복을 완전히 끝낼 때

※ continue, break는 for와 while 반복 구조 모두에서 사용 가능

확인 문제
문제 1 · 개념 확인
다음 중 반복 구조에 대한 설명으로 옳지 않은 것은?
① while 반복 구조는 조건식이 True인 동안 블록을 계속 반복 실행한다.
② for 반복 구조는 수열이나 리스트에 들어 있는 데이터의 개수만큼만 반복한다.
③ continue 명령이 실행되면 반복 블록의 나머지를 건너뛰고 반복이 완전히 종료된다.
④ break 명령이 실행되면 남은 반복 실행 작업이 모두 중단된다.
⑤ range(1, 6)은 1, 2, 3, 4, 5의 정수 수열을 만든다.
정답: ③
교과서 p.117 참고
continue는 현재 반복의 나머지 부분을 건너뛰고 다음 반복으로 이동한다. 반복이 완전히 종료되지는 않는다. 반복을 완전히 종료하는 것은 break이다.
