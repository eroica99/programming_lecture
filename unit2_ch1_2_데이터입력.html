1. 변수
교과서 pp.52–57
변수의 개념
p.52
변수 (variable)
값이 저장된 위치를 가리키는 특별한 이름이다. 변수를 활용하면 여러 가지 이름으로 데이터를 저장할 수 있으며, 필요할 때 편리하게 불러 재사용할 수 있다.
여러 가지 다른 값으로 바꾸어 저장할 수 있으므로 "변하는 수"라는 의미이다. 반면 정수, 실수, 문자 등 그 값 자체는 변할 수 없으므로 "항상 같은 수"라는 의미로 상수(constant)라고 부른다.
대입 연산자 (=)
p.53

변수에 원하는 값을 저장하기 위해 대입(assignment) 연산자 =를 사용한다.

형식설명
변수 = 값오른쪽의 값을 왼쪽의 변수에 저장한다.
주의 사항
• 변수 = 값 — 정상. 오른쪽의 값이나 계산 결괏값을 왼쪽 변수에 저장한다.
• 값 = 변수 — 오류 발생. 좌우 순서가 반드시 지켜져야 한다.
• 변수 = 변수 — 정상. 오른쪽 변수에 저장된 값을 왼쪽 변수에 저장한다.
코드
s = "Hello" print(s) print(s) print(s)
실행 결과
Hello Hello Hello

프로그램 실행 도중 새로운 값을 저장하면 이전 값을 덮어쓰며, 다시 바꾸기 전까지 그 값이 유지된다.

코드
n = 0 print(n) n = 1 print(n) print(n)
실행 결과
0 1 1
변수 이름 규칙
p.52
규칙가능한 예불가능한 예
영문자(A–Z, a–z), 숫자(0–9), 밑줄(_)만 사용한다. a, B1, _c, sum a+, a b, 2.0, 'c'
영문자 또는 밑줄(_)로 시작하며, 숫자로는 시작하지 않는다. n, _val, sum1 1n, 2a, 2score
영문자 대문자와 소문자를 구분하여 작성한다. ab, Ab, aB, AB는 모두 다른 변수로 인식된다.

의미와 처리 방법이 먼저 정해진 예약어나 내장 함수 이름은 변수 이름으로 사용할 수 없다.

구분설명예시
예약어 의미나 처리가 미리 정해져 있는 단어 if, while, for, continue, break 등
내장 함수 이름 처리 내용이 먼저 정해져 있는 함수 이름 print, int, float, input, min, max 등
리스트 변수와 인덱싱
pp.56–57
리스트 (list)
여러 개의 데이터를 하나의 변수에 순서대로 나열하여 저장하는 자료형이다. 대괄호([ ])와 쉼표(,)를 사용하여 만든다.
형식: 변수 = [값0, 값1, 값2, ...]

리스트에 저장된 각 값은 인덱스(참조 번호)로 접근한다. 첫 번째 값의 인덱스는 0번부터 시작한다.

1
d[0]
2
d[1]
3
d[2]
4
d[3]
5
d[4]
코드
d = [1, 2, 3, 4, 5] print(d) print(d[0], d[4]) print(d[4], d[3], d[2], d[1], d[0])
실행 결과
[1, 2, 3, 4, 5] 1 5 5 4 3 2 1

※ 리스트 전체를 출력하면 대괄호([ ])와 쉼표(,)가 포함되어 출력된다.

전역 변수 · 지역 변수 · 매개 변수
p.57
종류정의사용 가능 범위
전역 변수 함수 바깥(코드 블록 밖)에서 선언된 변수 프로그램의 모든 위치에서 참조·사용 가능
지역 변수 함수 안에서 선언된 변수 해당 함수 내부에서만 참조·사용 가능
매개 변수 함수 실행에 필요한 값을 함수 안으로 전달하기 위해 사용하는 특별한 변수 해당 함수 내부에서만 사용 가능

함수 안에서 전역 변수를 수정하려면 global 예약어를 반드시 선언해야 한다.

코드
s = 0 # 전역 변수 s def f(n): # 매개 변수 n global s k = 0 # 지역 변수 k while k < n: k = k + 1 s = s + k f(3) print(s)
실행 결과
6

※ global s 선언이 없으면 함수 안에서 전역 변수 s를 수정할 수 없어 오류가 발생한다.

2. input() 함수
교과서 pp.58–64
input() 기본
pp.58–59
input() 함수
키보드를 통하여 한 줄로 입력받은 내용을 문자열(str)로 반환하는 함수이다. 입력받은 값은 항상 문자열로 처리된다.
형식설명
변수 = input()키보드로 입력받은 한 줄 문자열을 변수에 저장한다.
변수 = int(input())입력받은 문자열을 정수로 변환하여 저장한다.
변수 = float(input())입력받은 문자열을 실수로 변환하여 저장한다.
코드
s = input() print(s)
입력
python
실행 결과
python
키보드로 123을 입력하면 문자열 "123"으로 처리된다. 정수로 사용하려면 반드시 int(input())으로 변환해야 한다.

input()은 한 번에 한 줄씩만 입력받으므로, 여러 줄을 입력받을 때는 여러 번 사용한다.

코드
s1 = input() s2 = input() print(s1) print(s2)
입력
hello world
실행 결과
hello world
split() — 한 줄 입력 나누기
pp.60–61, 63
split() 함수
입력받은 문자열을 특정 문자를 기준으로 나누어 분리한다. 기준 문자를 지정하지 않으면 빈칸(스페이스)을 기준으로 나눈다.
① 여러 변수에 나누어 저장

빈칸으로 구분된 두 값을 한 줄로 입력받아 각각 다른 변수에 저장한다.

코드
s1, s2 = input().split() print(s1) print(s2)
입력
hello world
실행 결과
hello world
② 리스트에 저장

빈칸으로 구분된 여러 값을 한 줄로 입력받아 하나의 리스트 변수에 저장한다.

코드
s = input().split() print(s[0]) print(s[1]) print(s[2])
입력
hello python world
실행 결과
hello python world
③ 특정 문자를 기준으로 나누기 — split('구분자')

split('구분자')를 사용하면 빈칸이 아닌 원하는 문자를 기준으로 나눌 수 있다.

코드
h, m, s = input().split(':') print(h) print(m) print(s)
입력
12:35:47
실행 결과
12 35 47
list(map(int, input().split())) — 정수 리스트 입력
pp.62–64

input().split()으로 나눈 값들은 모두 문자열이다. 정수로 사용하려면 map(int, ...)으로 변환한 후 list()로 리스트를 만든다.

구성 요소역할
input().split()한 줄 입력을 빈칸 기준으로 나누어 문자열 리스트로 반환
map(int, ...)리스트의 각 값을 정수(int)로 변환
list(...)변환된 값들을 리스트로 만들어 반환
코드
d = list(map(int, input().split())) print(d[0], d[1], d[2], d[3], d[4])
입력
1 2 3 4 5
실행 결과
1 2 3 4 5

구분자를 지정하면 다른 형식으로 입력받을 수도 있다.

코드
d = list(map(int, input().split(','))) print(d[0], d[1], d[2], d[3], d[4], sep='\n')
입력
1,2,3,4,5
실행 결과
1 2 3 4 5

문제 5
변수와 input() 함수에 대한 설명으로 옳지 않은 것은?
① 변수 이름은 영문자 또는 밑줄(_)로 시작해야 하며, 숫자로 시작할 수 없다.
② input() 함수로 입력받은 값은 항상 문자열로 처리된다.
③ 리스트의 첫 번째 요소는 인덱스 1번으로 접근한다.
④ 함수 안에서 전역 변수를 수정하려면 global 예약어를 사용해야 한다.
⑤ split()은 기준 문자를 지정하지 않으면 빈칸(스페이스)을 기준으로 나눈다.
정답: ③
교과서 p.56 참고
리스트의 첫 번째 요소는 인덱스 0번부터 시작한다. 인덱스 1번은 두 번째 요소를 가리킨다.

④ 전역 변수 수정 시 global 필요 — 옳음 (p.57)
