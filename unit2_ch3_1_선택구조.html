[프그 02-04] 순차, 선택, 반복의 제어 구조를 이해하고, 문제 상황에 알맞은 제어 구조를 선택하여 이를 적용한 프로그램을 작성할 수 있다.

1. 순차 구조와 선택 구조
교과서 p.99
순차 구조
프로그래밍 언어로 작성된 각각의 명령이 미리 정해진 규칙과 순서에 따라 위에서 아래로 한 줄씩 단계적으로 실행되는 구조이다.
선택 구조
어떤 조건을 검사한 결과에 따라 추가 작업이나 다른 작업을 실행하는 구조이다. 조건식의 계산 결과(True/False)에 따라 실행 흐름이 달라진다.
코드 블록 (code block)
코드들이 실행되는 논리적인 묶음이다. 파이썬에서는 일정한 간격의 들여쓰기를 사용하여 코드 블록을 표현한다.
※ 들여쓰기가 같은 코드는 같은 블록에 속하며, 선택·반복 구조에서 반드시 지켜야 한다.
2. if ~ 선택 구조
교과서 pp.99–101
형식과 동작 방식
p.99
형식
if 조건식: 실행 1 실행 2
동작
조건식 → True : 블록 실행 조건식 → False : 블록 건너뜀
필수 규칙
• if 조건식 끝에 콜론(:)이 반드시 있어야 한다.
• 실행할 코드는 반드시 들여쓰기(indent)로 작성한다.
• 들여쓰기가 없으면 if 블록에 속하지 않고 항상 실행된다.
예시 — 절댓값 출력
p.100
코드
n = int(input()) if n < 0: n = -n print(n)
입력
-5
결과
5

※ n이 0 이상이면 if 블록이 실행되지 않고 print(n)으로 바로 이동

예시 — 두 수 비교 (if 두 개)
p.101
코드
a = int(input()) b = int(input()) if a >= b: print(a) if a < b: print(b)
입력
3 5
결과
5

※ a>=b와 a<b는 서로 겹치는 부분이 없어 둘 중 하나는 반드시 True

3. if ~ else ~ 선택 구조
교과서 pp.102–104
형식과 동작 방식
p.102
형식
if 조건식: 실행 1 # True일 때 else: 실행 2 # False일 때
if 단독과의 차이
if 단독: True → 블록 실행 False → 건너뜀 if~else: True → if 블록 실행 False → else 블록 실행 (둘 중 하나 반드시 실행)
else: 끝에도 콜론(:)이 반드시 있어야 한다. else 블록에서는 조건식을 다시 검사할 필요가 없다. 조건이 False라는 것이 이미 확인되었기 때문이다.
예시 — 짝수/홀수 판별
p.103
코드
n = int(input()) if (n % 2) == 0: print("even") else: print("odd")
입력
3
결과
odd
예시 — 두 수의 차 (항상 양수로)
p.104
코드
a = int(input()) b = int(input()) if a >= b: print(a - b) else: print(b - a)
입력
3 5
결과
2
4. 선택 구조의 중첩
교과서 pp.105–108
개념
p.105
선택 구조의 중첩 (nested)
선택 구조 안에 다른 선택 구조를 넣어 여러 가지 복잡한 조건을 체계적으로 검사하거나 순서대로 검사하여 다르게 처리할 수 있다. if 블록이나 else 블록 안에 다른 if ~ else 구조를 자유롭게 중첩할 수 있다.
예시 — 성적 3단계 분류 (A/B/C)
pp.105–106
코드 (중첩 방식)
n = int(input()) if n >= 80: print("A") else: if n >= 60: print("B") else: print("C")
코드 (논리 연산자 방식)
n = int(input()) if n >= 80: print("A") if n < 80 and n >= 60: print("B") if n < 60: print("C")

※ 두 방식 모두 같은 결과. 중첩 방식은 이미 False인 조건을 다시 검사할 필요가 없어 더 효율적

입력 n조건 검사 흐름출력
90n≥80 → TrueA
70n≥80 → False → n≥60 → TrueB
50n≥80 → False → n≥60 → FalseC
예시 — 세 수 중 최댓값 찾기
p.107
코드
a = int(input()) b = int(input()) c = int(input()) if a >= b: if a >= c: print(a) else: print(c) else: if b >= c: print(b) else: print(c)
입력
-2 5 3
결과
5

※ a와 b 중 큰 수를 먼저 찾고, 그 수와 c를 비교하여 최댓값 결정

선택 구조 전체 비교
pp.99–108
구조조건 True조건 False특징
if 블록 실행 건너뜀 조건 만족 시에만 추가 실행
if ~ else if 블록 실행 else 블록 실행 둘 중 하나 반드시 실행
중첩 if 조건을 단계적으로 좁혀가며 검사 여러 경우를 체계적으로 분기
확인 문제
문제 1 · 개념 확인
파이썬의 선택 구조에 대한 설명으로 옳지 않은 것은?
① if 조건식 끝에는 반드시 콜론(:)을 붙여야 한다.
② if ~ else 구조에서 조건식이 False이면 else 블록이 실행된다.
③ if 단독 구조에서 조건이 False이면 if 블록은 실행되지 않고 건너뛴다.
④ 파이썬에서 코드 블록은 중괄호({ })로 구분한다.
⑤ 선택 구조 안에 다른 선택 구조를 중첩하여 사용할 수 있다.
정답: ④
교과서 p.99 참고
파이썬에서 코드 블록은 중괄호({ })가 아닌 들여쓰기(indent)로 구분한다. 이것이 파이썬의 가장 큰 특징 중 하나이다.
문제 2 · 빈칸 완성
빈칸 ㉠~㉢에 알맞은 말을 쓰시오.
if ~ else 구조에서 조건식이 ( ㉠ )인 경우에는 if 블록이 실행되고, ( ㉡ )인 경우에는 else 블록이 실행된다. 두 블록 중 ( ㉢ )가 반드시 실행된다.
㉠:
㉡:
㉢:
정답
교과서 p.102 참고
㉠ True   ㉡ False   ㉢ 하나(둘 중 하나)
if~else 구조는 조건이 True이든 False이든 반드시 어느 한 블록이 실행된다.
문제 3 · 개념 적용
다음 중 if ~ else 구조를 사용해야 적절한 상황은?
① 입력값이 음수인 경우에만 양수로 바꾸어 출력하고 싶을 때
② 두 수를 입력받아 더 큰 수를 출력하고 작은 수는 무시할 때
③ 입력값이 짝수인지 홀수인지 판별하여 각각 다른 문자열을 출력할 때
④ 점수가 100 초과일 때만 100으로 고쳐 출력하고 싶을 때
⑤ 입력값이 0보다 클 때만 "양수"를 출력하고 나머지는 출력하지 않을 때
정답: ③
교과서 pp.102–103 참고
if~else는 조건의 참·거짓 두 경우 모두 서로 다른 작업을 실행할 때 사용한다. ③번은 짝수일 때 "even", 홀수일 때 "odd"로 두 결과 중 하나를 반드시 출력해야 하므로 if~else가 적합하다.
①④⑤는 특정 조건에서만 추가 작업이 필요하므로 if 단독 구조가 더 적합하다.
