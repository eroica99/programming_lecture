1. 파일의 개념과 종류
교과서 pp.149–151
파일의 개념
p.149
파일 (file)
어떤 규칙이나 방법에 따라 저장 장치(storage)에 기록한 데이터의 집합이다. 파일 시스템을 통해서 체계적으로 저장·관리되며, 디렉터리(directory) 또는 폴더(folder)라고 부르는 계층 구조 안에 저장된다.
파일명의 확장자(filename extension)를 통하여 파일의 내용과 사용 목적을 대략 파악할 수 있다.
데이터 파일 vs 실행 파일
p.149
종류저장 내용특징
데이터 파일 프로그램 실행과 관련한 데이터와 레코드(record) 특별한 프로그램을 활용하여 읽고 쓸 수 있다.
실행 파일 컴퓨터를 직접 작동시키는 명령 다른 프로그램 없이 그 자체로 독립적으로 실행할 수 있다.

2. 파일 입출력 과정
교과서 pp.152–154
파일 작업 순서
p.153 [표Ⅱ-14]
쓰기 작업:
파일 열기 open() → 데이터 쓰기 write() → 파일 닫기 close()
읽기 작업:
파일 열기 open() → 데이터 읽기 read() → 파일 닫기 close()
작업설명주의 사항
파일 열기
open()		 파일이 저장된 위치를 프로그램이 접근할 수 있도록 준비한다. 한 파일에 한 번에 하나의 프로그램만 접근 가능
파일 읽기
read() / readline()		 파일에 저장되어 있는 내용을 프로그램으로 읽어 들인다.
· read(): 파일 전체 데이터를 문자열 하나로 읽음
· readline(): 한 줄씩 텍스트를 읽어 들임		 파일이 없으면 오류 발생
파일 쓰기
write()		 파일에 데이터를 기록한다. 문자열만 기록 가능하며, 수는 str()로 변환해야 한다. 줄 바꿈이 필요하면 "\n" 직접 추가
파일 닫기
close()		 프로그램과 파일의 연결을 끊는다. 파일을 닫아야만 파일이 완성됨
파일 열기 모드
p.153 [표Ⅱ-15]
모드의미설명
r파일 읽기해당 파일이 있어야 한다. 없으면 오류 발생.
w파일 쓰기파일이 이미 있는 경우는 덮어쓰기가 된다.
x파일 쓰기해당 파일이 없는 경우에만 파일이 생성된다.
a파일 추가해당 파일의 마지막에 데이터를 추가한다. 파일이 없는 경우에는 새로 생성하여 데이터를 추가한다.
b바이너리 모드2진수 코드로 데이터를 읽고 쓴다.
t텍스트 모드텍스트 코드로 데이터를 읽고 쓴다.
+업데이트 모드기록되어 있는 데이터를 읽고 업데이트하여 쓸 수 있다.

예시 ① — 단어 두 개를 한 줄씩 저장

코드
f = open("out.txt", "w") n = int(input()) for i in range(n): d = int(input()) f.write(str(d)) f.write("\n") f.close()

3 123 456 789
out.txt 내용
123 456 789

※ write()는 문자열만 기록 → 정수 d를 str(d)로 변환 필수

TXT 파일 읽기
pp.158–160
예시 ③ — TXT 파일 전체 읽어 출력
코드
f = open("in.txt", "r") rows = (f.read()).split("\n") f.close() for row in rows: print(row)
in.txt 내용
hello python world good
결과
hello python world good

※ f.read() → 파일 전체를 문자열 하나로 읽음 → split("\n")으로 줄 단위 리스트 생성

예시 ④ — TXT 파일에서 앞 n개만 정수로 출력
코드
f = open("in.txt", "r") rows = (f.read()).split("\n") f.close() n = int(input()) for row in rows[:n]: print(int(row))
in.txt / 입력
2 13 167 1009
입력 3 → 결과
2 13 167

※ rows[:n] — 인덱스 0번부터 n-1번까지의 부분 리스트  ·  파일에서 읽은 값은 문자열이므로 int() 변환 필수

예시 ⑤ — readline()으로 한 줄씩 읽기

readline()은 파일에서 한 줄씩 텍스트를 읽어 들이는 함수이다. 반복문과 함께 사용하여 여러 줄을 순서대로 처리할 때 유용하다. (p.152)

코드
f = open("in.txt", "r") n = int(input()) for i in range(n): line = f.readline() print(int(line)) f.close()
in.txt / 입력
10 20 30 40 50
입력 3 → 결과
10 20 30
구분read()readline()
읽는 단위파일 전체를 문자열 하나로한 줄씩 순서대로
활용 방식.split("\n")으로 줄 단위 분리반복문 안에서 한 줄씩 호출
교과서 예시p.158 (전체 읽기)p.152 개념, p.167 프로젝트

※ readline()은 읽은 줄 끝에 "\n"이 포함되어 있으므로 정수 변환 시 int(line)처럼 사용하면 자동으로 처리된다.

핵심 주의 사항
pp.158–160
① 읽기 모드("r")로 열 때 해당 파일이 존재하지 않으면 오류가 발생한다. 프로그램과 같은 위치에 파일이 있어야 한다.
② 파일에서 읽은 값은 모두 문자열(str)이다. 정수로 사용하려면 int(), 실수로 사용하려면 float()으로 변환해야 한다.
③ 쓰기 모드("w")로 열 때 동일한 파일명이 이미 있으면 덮어쓰기가 된다.
4. CSV 파일 처리
교과서 pp.161–164
CSV 파일 (Comma Separated Values)
데이터들 사이에 쉼표(,)를 넣어 저장하는 텍스트 파일이다. 대부분의 스프레드시트 프로그램·데이터베이스 프로그램과 호환되며, 빅데이터·데이터 마이닝 분야에서 대량의 데이터를 다루기 위해 자주 사용된다.
CSV 파일 읽기
pp.161–162
코드
f = open("data.csv", "r", encoding="utf-8-sig") rows = (f.read()).split("\n") f.close() for row in rows: values = row.split(",") for value in values: print(value, end=" ") print()
data.csv 내용
이름,국어,수학 김철수,90,85 이영희,80,95
결과
이름 국어 수학 김철수 90 85 이영희 80 95
코드 부분역할
encoding="utf-8-sig"CSV 파일 앞의 BOM(Byte Order Mark) 값을 자동 제거하여 첫 글자가 깨지지 않게 한다.
rows = (f.read()).split("\n")파일 전체를 읽어 줄 단위로 분리 → 리스트
values = row.split(",")각 줄을 쉼표 기준으로 분리 → 리스트
print(value, end=" ")스페이스로 구분하여 한 줄에 출력
print()한 행 출력 후 줄 바꿈
CSV 파일 쓰기
pp.163–164
코드
rows = [["data A","data B","sum","mean"], [1, 2, 3, 1.5], [3, 4, 7, 3.5], [5, 6, 11, 5.5]] f = open("dataout.csv", "w") for row in rows: for value in row[:-1]: f.write(str(value)+",") f.write(str(row[-1])+"\n") f.close()
생성된 dataout.csv 내용
data A,data B,sum,mean 1,2,3,1.5 3,4,7,3.5 5,6,11,5.5
코드 부분역할
row[:-1]리스트의 처음부터 마지막 값 바로 앞까지의 부분 리스트 → 쉼표와 함께 출력
row[-1]리스트의 마지막 값 → 쉼표 없이 줄 바꿈과 함께 출력
str(value)+","수 데이터를 문자열로 변환 후 쉼표 추가하여 기록
str(row[-1])+"\n"마지막 값을 문자열로 변환 후 줄 바꿈 추가
write()는 문자열만 기록 가능하다. 정수·실수 등 수 데이터는 반드시 str()로 변환한 후 write()로 기록해야 한다.

확인 문제
문제 1 · 개념 확인
파일에 대한 설명으로 옳지 않은 것은?
① 파일은 어떤 규칙이나 방법에 따라 저장 장치에 기록한 데이터의 집합이다.
② 파일명 확장자를 통하여 파일의 내용과 사용 목적을 대략 파악할 수 있다.
③ 파일에 데이터를 기록한 후 close()를 호출하지 않아도 파일이 완성된다.
④ TXT 파일은 ASCII나 UTF-8 문자 코드를 사용해 작성할 수 있다.
⑤ CSV 파일은 데이터들 사이에 쉼표(,)를 넣어 저장하는 텍스트 파일이다.
정답: ③
교과서 p.153 [표Ⅱ-14] 참고
파일에 데이터를 기록하고 있다면 파일 쓰기를 종료(close())해야만 파일이 만들어진다. close()를 호출하지 않으면 파일이 완성되지 않을 수 있다.
