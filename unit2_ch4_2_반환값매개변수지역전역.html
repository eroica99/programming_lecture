1. 반환값과 매개변수
교과서 pp.135–139
함수의 4가지 형태
p.135 [표Ⅱ-11]

함수는 반환값(return value)의 유무와 매개변수(parameter)의 유무에 따라 4가지로 구분된다.

반환값 없음 · 매개변수 없음
def f(): print("Hello") f()
반환값 없음 · 매개변수 있음
def f(n): print(n) f(3)
반환값 있음 · 매개변수 없음
def f(): return 3 print(f())
반환값 있음 · 매개변수 있음
def f(a, b): return a + b print(f(3, 5))
return — 반환값과 복귀
pp.135–136
반환값 (return value)
함수 실행이 완료된 후 함수가 호출되었던 위치로 되돌려 놓는 값이다. return 명령 뒤에 값을 작성하여 반환한다.
return이 없거나 return만 있으면 None이 반환된다.
① return — 복귀만 (값 없음)

return만 쓰면 값 없이 호출 위치로 복귀한다. return 이후 코드는 실행되지 않는다.

코드
def f(): print("Hello") return print("World") # 실행 안 됨 f()
결과
Hello
② return 값 — 값을 반환하며 복귀

return 값을 쓰면 함수가 호출된 위치에 그 값을 되돌려 놓으며 복귀한다.

코드
def f(): return 3 print(f())
결과
3

※ print(f())에서 f()가 먼저 실행되어 3을 반환 → print(3) 실행

return이 실행되면 함수의 나머지 부분은 실행되지 않고 즉시 호출 위치로 복귀한다. return 이후에 작성된 코드는 문법적으로 오류가 아니지만 절대 실행되지 않는다.
매개변수 (parameter)
pp.136–139
매개변수 (parameter)
함수 실행에 필요한 값을 함수 안으로 전달하기 위해 사용하는 특별한 변수이다. 함수 정의 시 소괄호 안에 순서대로 작성한다. 매개변수는 해당 함수 내부에서만 사용할 수 있는 특별한 형태의 지역 변수이다.
함수를 호출할 때 소괄호 안에 쉼표로 나열한 값을 전달 인자(argument)라고 한다.
매개변수 1개
코드
def f(n): print(n) f(3)
결과
3

f(3) 호출 시 매개변수 n에 3이 저장됨

매개변수 2개 + return
코드
def f(a, b): return a + b print(f(3, 5))
결과
8

f(3, 5) 호출 시 a=3, b=5 순서대로 저장 → a+b=8 반환

함수 안에서 선택 구조 사용 — 두 수 중 작은 값 반환
코드
def f(a, b): if a < b: return a else: return b print(f(3, 5))
f(3,5) 흐름
a=3, b=5 3 < 5 → True return 3 → 3 반환
결과
3

※ 두 값이 같은 경우(a==b)는 else 실행 → b 반환 (p.139)

2. 지역 변수와 전역 변수
교과서 pp.140–143
지역 변수와 전역 변수의 차이
pp.140–142
종류선언 위치사용 가능 범위특징
지역 변수 함수 코드 블록 안에서 선언 해당 함수 내부에서만 참조·사용 가능 함수 실행이 끝나면 사라짐
전역 변수 함수 코드 블록 밖(가장 바깥쪽)에서 선언 프로그램의 모든 위치에서 참조·사용 가능 함수 안에서 수정하려면 global 필요
매개변수 함수 정의 시 소괄호 안에 선언 해당 함수 내부에서만 사용 가능 특별한 형태의 지역 변수
지역 변수 사용 예시
pp.140–141
코드
def f(n): k = 0 # 지역 변수 k s = 0 # 지역 변수 s while k < n: k = k + 1 s = s + k return s print(f(3))
변수 추적 (n=3)
k=0, s=0 k=1, s=1 k=2, s=3 k=3, s=6 → return 6
결과
6

※ k, s는 함수 f 내부에서만 사용 가능한 지역 변수. 함수 밖에서 k나 s에 접근하면 오류 발생

전역 변수 + global 키워드
p.142

함수 안에서 전역 변수를 읽는 것은 가능하지만, 수정(값 변경)하려면 global 예약어를 반드시 선언해야 한다.

코드
s = 0 # 전역 변수 s def f(n): global s # 전역 변수 s 사용 선언 k = 0 # 지역 변수 k while k < n: k = k + 1 s = s + k f(3) print(s)
변수 추적
전역 s = 0 f(3) 호출 k=0→1, s=0+1=1 k=1→2, s=1+2=3 k=2→3, s=3+3=6 전역 s = 6
결과
6
global s 선언이 없으면 함수 안에서 s = s + k를 실행할 때 "변수 s에 어떤 값이 저장되어 있는지 모르는 상태에서 연산"이 실행되므로 오류가 발생하고 프로그램이 중단된다.
매개변수와 전역 변수 이름이 같은 경우
p.143
코드
def f(n): # 매개변수 n s = 0 for k in range(1, n+1): s = s + k return s n = int(input()) # 전역 변수 n print(f(n))
설명
전역 변수 n과 매개변수 n은 이름은 같지만 서로 다른 변수이다. 전역 n: 입력받은 값 저장 매개변수 n: 함수 내부에서만 사용되는 지역 변수

※ 입력 3 → 전역 n=3 → f(3) 호출 → 매개변수 n=3 → 1+2+3=6 반환 → print(6)

함수 핵심 개념 전체 정리
p.144 요약
개념설명
def함수를 정의하는 예약어. 함수 정의는 호출보다 반드시 먼저 작성
함수 호출함수 이름()으로 정의된 코드 블록을 실행시키는 것
return① 복귀만 / ② 값을 반환하며 복귀. return 이후 코드는 실행 안 됨
매개변수함수에 전달할 값을 받는 변수. 소괄호 안에 선언. 지역 변수의 일종
지역 변수함수 안에서 선언. 함수 내부에서만 사용 가능
전역 변수함수 밖(가장 바깥쪽)에서 선언. 모든 위치에서 사용 가능
global함수 안에서 전역 변수를 수정할 때 반드시 선언해야 하는 예약어
확인 문제
문제 1 · 개념 확인
return과 매개변수에 대한 설명으로 옳지 않은 것은?
① return 명령이 실행되면 함수의 실행이 중단되고 호출 위치로 복귀한다.
② return 명령 뒤에 값을 쓰면 그 값을 함수가 호출된 위치에 돌려준다.
③ 매개변수는 함수 정의 시 소괄호 안에 작성하는 특별한 형태의 지역 변수이다.
④ return이 없는 함수는 호출하면 오류가 발생한다.
⑤ 함수에 여러 값을 전달할 때 쉼표로 나열하면 순서대로 매개변수에 저장된다.
정답: ④
교과서 p.135 참고
return이 없는 함수도 정상적으로 호출되어 실행된다. return이 없으면 함수 블록이 끝날 때 자동으로 복귀(None 반환)되는 것과 같다. 오류가 발생하지 않는다.

문제 7 · 응용 — 함수 설계
두 개의 정수를 매개변수로 전달받아, 두 수 중 큰 값을 반환하는 함수 bigger(a, b)를 작성하시오. 그 후 세 수(n, m, k)를 입력받아 세 수 중 최댓값을 출력하는 코드를 완성하시오.
힌트: bigger(bigger(n, m), k)를 활용
정답
